Unit 4: Giving Feedback

Strong Teams: Feedback, Communication & Performance Improvement | Year 8 Health and Physical Education

Learning intention: Students practise and apply specialised movement skills in a variety of movement situations, using observation, peer feedback and reflection to improve control, accuracy, communication and team performance.

Rationale

This unit builds the practical and social skills students need to improve performance in games. Students learn that feedback is not criticism; it is information that helps a person know what to keep doing and what to try next. The weekly structure can stay within one focus sport for several lessons so students build individual skills first, then combine those skills into simple tactics and team decisions. Other sports and modified games can be used as comparison examples or later transfer tasks.

Assessment Overview

Main task: Students create a Google Slides folio / digital coaching portfolio across the unit. The folio includes annotated movement skill slides, key coaching points, common mistakes, helpful feedback examples, peer feedback records, personal reflection, teamwork and communication reflection, and a final performance improvement reflection.

  • Google Slides Folio / Performance Improvement Analysis
  • Peer feedback task
  • Short written reflections
  • Teacher observation of participation, communication, leadership, fair play and teamwork

Success Criteria

  • I can explain key coaching points for different movement skills.
  • I can identify common mistakes and suggest a useful correction.
  • I can use observation to analyse my own and others' performance.
  • I can give feedback that is specific, respectful and helpful.
  • I can receive feedback calmly and use it to improve control and accuracy.
  • I can explain how my communication improves the activity or affects team performance.
  • I can reflect on my own performance, communication and teamwork.
  • I can support classmates with different skill levels and keep games inclusive.
  • I can suggest simple modifications that make games safer, fairer or more inclusive.

Practical Teamwork Scoring Ideas

Add communication points to practical activities so students see that positive team culture is part of success, not an optional extra.

  • +1 encouragement point for positive language after a mistake.
  • +1 teamwork point for helping a teammate reset or rejoin the game.
  • +1 coaching point for giving a specific, respectful cue.
  • +1 leadership point for including someone or organising fairly.
  • +1 respect point for disagreeing calmly about a call.

Suggested Weekly Structure

Recommended model: choose one main focus sport for the block, then teach several connected skills before moving into tactics. Tennis, soccer, basketball or volleyball can all work. The examples below use soccer as the main sport because it clearly builds from passing and receiving into space, support and team tactics.

Week 1Focus sport introduction: safety, fair play, respectful feedback language, baseline skill check and one core skill such as inside foot pass.Start folio slides 1-2; introduce feedback sentence starters and personal goal.
Week 2Skill development: refine core technique, add receiving/first touch, control, accuracy and partner observation.Complete first skill annotation and feedback reflection; collect peer feedback evidence.
Week 3Skill extension: add a second connected skill such as dribbling, shielding, movement into space, shooting or defending.Complete second skill annotation; compare visual, auditory and kinaesthetic feedback.
Week 4Tactical development: small-sided games, creating space, support play, decision-making, attacking and defending roles.Reflection on how feedback improved control, accuracy and tactical choices.
Week 5Team performance: apply skills and tactics in modified games with communication, emotional regulation, inclusion and fair play focus.Complete peer feedback record and teamwork/communication reflection; trial one rule, role or equipment modification.
Week 6Performance improvement review: revisit baseline skill, apply feedback, demonstrate improvement and explain next steps.Complete final performance improvement reflection; submit folio and observation evidence.

Alternative focus sport options: Tennis could build from grip and forehand into backhand, serve/underarm feed, court positioning and rally tactics. Basketball could build from passing and catching into shooting, dribbling, spacing and give-and-go play. Volleyball could build from ready position and dig into set, serve receive, court coverage and three-touch tactics.

Teacher Notes for This Class Context

  • Use one main sport long enough for students to experience a real improvement cycle: practise, receive feedback, adjust, apply in a game, reflect.
  • Use mixed-skill pairings carefully: pair confident performers with students who can explain respectfully, not simply the strongest players.
  • Pre-teach the difference between helpful feedback and unhelpful comments before competitive games begin.
  • Name unhelpful behaviours directly: yelling after mistakes, blaming, sarcasm, arguing over calls, refusing to pass and negative body language.
  • Pause games when frustration rises and reset with a short prompt: "What is one useful cue we can give now?"
  • Reward communication behaviours as visibly as skill performance: encouragement, calm repair after mistakes, including others and listening to feedback.
  • Keep reflections short and frequent so students connect feedback to actual improvement, not just end-of-unit writing.
