# Year 8 HPE Giving Feedback Resource Pack

Ready-to-use assessment and teaching resources for:

**Giving Feedback**  
**Strong Teams: Feedback, Communication & Performance Improvement**

Open `index.html` to access the full teacher resource pack.

Open `student-copies/index.html` to access the student-facing resource pack.

## Main Focus

Students practise movement skills, use feedback to improve performance, and reflect on how positive communication makes games more enjoyable and successful.

## Includes

- Teacher unit overview
- Scope, sequence and SA Education dispositions alignment
- Student assessment task
- Google Slides folio guide
- Peer feedback sheet
- Reflection exit slips
- Teacher observation checklist
- Assessment rubric
- Annotated forehand and soccer pass examples
- Student-only resource copies
